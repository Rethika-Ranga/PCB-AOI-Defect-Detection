"AI-Powered PCB Defect Detection and Automated Optical Inspection System"
      An end-to-end Automated Optical Inspection (AOI) system for Printed Circuit Boards (PCBs) using YOLOv8, inspired by industrial quality inspection workflows. The system performs real-time defect detection, calculates PCB quality scores, generates PASS/REVIEW/FAIL decisions, and automatically creates inspection reports.
Defects Detected:
      ~ Missing Hole
      ~ Mouse Bite
      ~ Open Circuit
      ~ Short
      ~ Spur
      ~ Spurious Copper
Features:
      ~ Detects 6 common PCB defects
      ~ Real-time inspection using webcam
      ~ YOLOv8-based object detection
      ~ Quality score calculation
      ~ PASS / REVIEW / FAIL decision logic
      ~ Defect severity classification
      ~ Automated PDF inspection report generation
      ~ Timestamped board inspection records
      ~ Snapshot capture during live inspection
Tech Stack:
      ~ Python
      ~ YOLOv8
      ~ Ultralytics
      ~ OpenCV
      ~ PyTorch
      ~ FPDF2
      ~ NumPy
Model Performance:
      ~ Precision: 90.2%
      ~ Recall: 81.8%
      ~ mAP50: 86.9%
Key Learnings:
This project provided practical experience in:
      ~ Computer Vision
      ~ Object Detection with YOLOv8
      ~ Industrial quality inspection workflows
      ~ PCB defect analysis
      ~ OpenCV
      ~ Real-time inference
      ~ Report generation and automation
